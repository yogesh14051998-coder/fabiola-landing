## Features

- Responsive design — mobile, tablet, desktop
- Champagne gold / white alternating section layout
- Scroll animations — re-trigger every time sections enter the viewport
- Hero parallax background effect
- Area cards with click-to-enlarge lightbox → scrolls to contact form
- Contact form sends inquiry via email client + WhatsApp simultaneously
- Social icons with labels: Phone (Apple green), Email (Apple blue), WhatsApp, Instagram
- Staggered card animations on scroll
