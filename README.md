# Fabiola Acevedo P.A. — Luxury Real Estate Landing Page

> Live site: [fabiola-landing-omega.vercel.app](https://fabiola-landing-omega.vercel.app/)

A luxury real estate landing page for **Fabiola Acevedo P.A.**, a South Florida luxury real estate advisor with 25+ years of experience across Palm Beach, Miami, Fort Lauderdale, and surrounding markets.

---

## Features

- Elegant navy & gold design matching the "Team Luxury by Fab" brand
- Fully responsive — mobile, tablet, and desktop
- Smooth scroll navigation with fixed header
- Hero section with consultation & WhatsApp CTAs
- About section with professional headshot
- Areas of focus grid (Palm Beach, Miami, Fort Lauderdale, Boca Raton, and more)
- Services section (Buy, Sell, Pre-Construction, Relocate, Invest)
- Contact form with inquiry type selector
- Auto-deploys to Vercel on every push

---

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no dependencies
- Google Fonts (Cormorant Garamond + Montserrat)
- Hosted on Vercel, source on GitHub

---

## Deployment

Push to `master` and Vercel auto-deploys.

```bash
git add .
git commit -m "your message"
git push
```

---

## Customization

Open `index.html` and replace the following placeholders:

| Placeholder | Replace with |
|---|---|
| `[Your Phone Number]` | Fabiola's phone number |
| `[Your Email Address]` | Fabiola's email address |
| `[Your Website Link]` | Brokerage website URL |
| `wa.me/1XXXXXXXXXX` | WhatsApp number |

To swap the photo, replace `fabiola.jpeg` in the root folder.
